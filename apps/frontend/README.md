# Frontend (Next.js)

Stack sugerido: Next.js + TailwindCSS (opcional) + TypeScript.

Comandos iniciais:
- Instalar: `npm install` ou `pnpm install`
- Rodar dev: `npm run dev`

Páginas a implementar inicialmente:
- `/` — catálogo público
- `/items/[slug]` — página do item
- `/reservar` — fluxo de reserva
- `/admin/*` — área administrativa (pode ser rota separada ou app independente)
