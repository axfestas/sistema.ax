# Style Guide - Ax Festas

Breve guia para início do design system.

Tokens:
- Fontes: `Lucky Bones` (AX logo), `Open Sans` (texto), `Autun` (títulos)
- Cores: ver `design/tokens.json` e `design/tokens.css`

Componentes iniciais a criar:
- Button (primary, secondary, outline)
- Card (item card para catálogo)
- Badge (status de reserva)
- Modal (reserva rápida)

Ações recomendadas:
- Implementar com TailwindCSS + design tokens, ou com CSS-in-JS (Styled Components, Emotion)
- Adicionar Storybook para documentação de componentes
