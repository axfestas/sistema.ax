## Paleta de Cores

| Nome | HEX | RGB | CMYK |
|---|---|---|---|
| Amarelo Dourado | `#FFC107` | 255, 193, 7 | 0, 24, 97, 0 |
| Cinza Grafite | `#4A4A4A` | 74, 74, 74 | 0, 0, 0, 71 |
| Azul Sereno | `#88A9C3` | 136, 169, 195 | 30, 13, 0, 24 |
| Rosa Vibrante | `#EF3A5D` | 239, 58, 93 | 0, 76, 61, 6 |
| Branco Neve | `#FFFFFF` | 255, 255, 255 | 0, 0, 0, 0 |

## Uso recomendado
- **Amarelo Dourado**: CTAs, botões primários e destaques.
- **Cinza Grafite**: textos, títulos e elementos neutros.
- **Azul Sereno**: backgrounds, headers e componentes suaves.
- **Rosa Vibrante**: acentos e alertas.
- **Branco Neve**: fundos e cards.
